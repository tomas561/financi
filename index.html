⚙️ Configuração de Nuvem
Como configurar: edite o arquivo HTML e substitua os valores placeholder nas constantes FIREBASE_CONFIG e GOOGLE_CLIENT_ID. Veja o guia abaixo.
🔥 Firebase Realtime Database (sync gratuito entre máquinas)
  1. Acesse console.firebase.google.com → projeto monosup-financi
  2. No menu esquerdo: Build → Realtime Database
  3. Clique "Criar banco de dados" → região United States → "Iniciar no modo de teste" → Ativar
  4. Copie a URL que aparece (ex: https://monosup-financi-default-rtdb.firebaseio.com)
  5. Cole a URL na constante databaseURL dentro do <script type="module"> no <head> do HTML
✅ O Realtime Database é 100% gratuito no plano Spark — sem necessidade de cartão de crédito.
📁 Google Drive (upload de NFs)
  1. Acesse console.cloud.google.com
  2. Crie ou selecione um projeto → APIs & Services → Enable APIs → ative Google Drive API
  3. Em Credentials → Create Credentials → OAuth 2.0 Client ID
  4. Tipo: Web application
  5. Em Authorized JavaScript origins adicione exatamente:
    https://tomas561.github.io
  6. Copie o Client ID e cole na constante GOOGLE_CLIENT_ID no script principal
  7. Clique em "🔑 Conectar Google" no topo da ferramenta antes de anexar NFs
